# Cloudant API mocha test suite
A test suite that tests Cloudant's suite of features for the purpose of keeping abreast of the latest API changes. Uses the official npm module where applicable.

The tests are structured based on the official [IBM Cloudant Documentation](https://docs.cloudant.com/index.html) resource, in particular following the sequential layout of the [IBM Cloudant Documentation](https://docs.cloudant.com/api.html#-api-reference) section.

To run the tests, clone this repo:
```
cd [your app root]
git clone https://github.com/brilliantmichael/cloudant-api-mocha-tests.git
npm install
```
- then run:
```
npm start
```
